Detailed Description

#include <QmlViewManager.h>

The View::QmlViewManager class manages the QML view. It links services, widgets and models to the view

It is a singleton class that hold the various singleton managers needed by the modules of the Tepee3D application as well as the view properties. Also, this is where qml models and utility classes are exposed to the Qml Engine.

See also View::QmlViewProperties.
