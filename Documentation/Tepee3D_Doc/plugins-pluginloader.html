PluginLoader Class

(Plugins::PluginLoader)

The Plugins::PluginLoader class is an utility static class aimed at easing the process of loading plugin libraries. It loads all the local plugin libraries in a list. That list can later be used to instantiate new plugin instances. More...

This class was introduced in Tepee3D 1.0.

Detailed Description

The Plugins::PluginLoader class is an utility static class aimed at easing the process of loading plugin libraries. It loads all the local plugin libraries in a list. That list can later be used to instantiate new plugin instances.
