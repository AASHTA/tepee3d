Detailed Description

#include <ListModel.h>

The Models::ListModel class is a utility model class

The Models::ListModel class provides an easy way to create a C++ model and later expose it to the Qml View.

See also Models::SubListedListModel.

Property Documentation

count : const int

This property holds the number of rows of the model.

bool ListModel::removeRow(int row, const QModelIndex & index = QModelIndex())

Removes a single row at position defined by row . The index argument is optional. Returns true if row was removed, false if row not found or row is invalid.

bool ListModel::removeRows(int row, int count, const QModelIndex & index = QModelIndex())

Removes several rows starting at position defined by row until either count or the model's last row is reached. The index argument is optional. Returns true if the rows were removed, false if row is invalid.
