Detailed Description

#include <BasicRoom.h>

The BasicRoom::BasicRoom class is simplest implementation of the RoomBase

The BasicRoom class represents a squared room. It is compiled in a shared library and then dynamically loaded in the Tepee3DEngine at runtime.

See also Room::RoomBase.
